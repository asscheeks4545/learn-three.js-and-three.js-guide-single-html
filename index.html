How this course works

Three ways to code

The editor opens empty on purpose — write the code yourself. Stuck? Use Load Example for the full solution, or Fill-In Exercise for a scaffold with a few TODO blanks to solve.

Fill-In Helper

When you load a fill-in exercise, a helper panel appears with the goal, the concepts you already know (from this + previous chapters), and a checklist of blanks. Reveal an answer only if you are truly stuck.

Shortcuts

Ctrl+↵ Run code  ·  Alt+←/Alt+→ Prev/Next lesson  ·  Tab Insert two spaces

Your code and progress are saved automatically in your browser. Reload the page and you will pick up right where you left off.
